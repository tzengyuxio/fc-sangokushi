# ROM_Kanji 與 [EXT]Name 差異對照表

共發現 **83** 筆差異

## 差異原因分析

1. **Tile ID 對照錯誤**: KANJI_TILE_MAP 中的對應不正確
2. **Tile ID 缺少對照**: 某些 tile ID 尚未加入 KANJI_TILE_MAP
3. **同音異字**: 同一個 tile ID 可能在不同武將代表不同漢字（遊戲限制）

### 關於同音異字

FC 版三國志因 ROM 容量限制，漢字 tile 數量有限（約 238 個）。
部分同音字共用相同的 tile ID，導致解碼結果與實際姓名不符。

例如：
- Tile ID 用於「趙」的位置，實際顯示可能是「夏」的圖形
- 這是遊戲設計上的取捨，非解析錯誤

## 結論

KANJI_TILE_MAP 的對照主要是根據已知武將進行推斷，但由於：

1. **漢字數量限制**: FC ROM 容量有限，無法收錄所有漢字
2. **同音字共用**: 多個同音字可能共用同一個 tile 圖形
3. **初始推斷誤差**: 部分 tile ID 的對照是根據有限樣本推測

建議：
- 此對照表可作為「基於 ROM 解析的近似姓名」
- 精確姓名應以 [EXT]Name（外部資料）為準
- 若需修正 KANJI_TILE_MAP，需要實際查看 ROM 中的 tile 圖形確認
