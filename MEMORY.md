## ROM 概要

| 項目 | 值 |
|------|------|
| 檔名 | `Sangokushi (Japan).nes` |
| Mapper | 1 (MMC1) |
| PRG ROM | 256 KB = 16 banks × 16 KB |
| PRG 偏移 | 0x00010 – 0x4000F |
| CHR ROM | 0 KB (CHR-RAM) |
| Bank 換算 | `file_offset = bank × 0x4000 + 0x10` |
